Trajectory tracking controller

Having a KUKA LBR4 robot in my current lab, I wanted to setup a ROS interface to run joint space trajectories on the robot. The goal is to given a joint space position trajectory, the robot should track the trajectory accurately. This joint space trajectory can be obtained from motion planners, a simple interface would be to use move-it to obtain a joint space trajectory.
