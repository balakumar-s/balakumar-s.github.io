Robots can be complex. Complexity arises from the large dimensional states and inputs that a robot can have. The lack of a good model to map inputs to outputs and also to completing tasks. The Darpa Robotics Challenge(DRC) is an example of the hurdles in achieving human-like performance in robots.

Robots outperform humans in model based environments.
Achieving this precision in the real world riddled with model uncertainties requires exploration into machine learning. This blog attempts to explore machine learning concepts and how to use them to achieve tasks with robots.

Robots have states and actions. Actions affect a change in state. All states might not be observable at run time. A task might not have an analytical solution using the states and actions of the robot.
