Policy Search

Consider a robot with states $x$ and inputs $u$. A policy $\pi(x,u)$ can be used to obtain the next best action $u_{t}=\pi(u_t|x_t)$ that maximizes some long term expected reward $R(x,u)$. The subscript $t$ refers to the time-step. A feedback controller also allows for getting the best $u_{t}$. The policy can also get the next state given the current state and the input $x_{t+1}=\pi(x_{t+1}|x_t,u_{t})$.
